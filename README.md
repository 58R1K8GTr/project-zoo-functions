# 🦁 Zoo Functions

O **Zoo Functions** é um projeto desenvolvido em **JavaScript puro (ES6)** focado na organização e busca de dados para o gerenciamento de um zoológico.

Neste projeto, eu trabalhei no desenvolvimento de algoritmos utilizando funções de alta ordem (Higher Order Functions) para processar dados sobre animais e pessoas colaboradoras, além de implementar testes unitários com o framework Jest para garantir a qualidade do código.

---

## 🚀 O que foi desenvolvido

A aplicação consiste em um conjunto de funções de lógica de negócio que resolvem problemas como:
*   **getSpeciesByIds:** Busca espécies de animais através de um ou mais IDs.
*   **getAnimalsOlderThan:** Valida se todos os animais de uma espécie possuem uma idade mínima.
*   **getEmployeeByName:** Localiza pessoas colaboradoras pelo primeiro ou último nome.
*   **getRelatedEmployees:** Identifica gerentes e lista as pessoas que estão sob sua liderança.
*   **countAnimals:** Contabiliza o número de animais residentes por espécie ou sexo.
*   **calculateEntry:** Calcula o valor monetário total das entradas do zoológico com base na faixa etária dos visitantes.
*   **Testes Unitários:** Criação de cobertura de testes automatizados com Jest para as funções `handlerElephants` e `getOpeningHours`.

---

## 🛠️ Tecnologias Utilizadas

*   **JavaScript (ES6):** Uso avançado de métodos de array como `.map()`, `.filter()`, `.reduce()` e `.find()`.
*   **Node.js (v16):** Ambiente de execução local da aplicação.
*   **Jest:** Framework para a execução e cobertura de testes unitários.
*   **ESLint:** Ferramenta de análise estática para garantia das boas práticas de código.
